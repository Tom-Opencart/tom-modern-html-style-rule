Panels

Module Purpose

Framed content panels with accent top-border and tick markers. For feature lists and structured information.

Key Features

  • Automation: key binding on order status transition to "Completed".
  • Uniqueness control: same code never issued twice until admin returns it to pool.
  • Stock sync: product quantity auto-recalculates from free key count.
  • Admin convenience: manual entry, CSV import, status view, and key management in order.
